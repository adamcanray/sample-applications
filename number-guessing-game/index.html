Welcome to the Obscuro Number Guessing Game!

The game contract contains a secret number, generated when the contract was deployed, and visible only within the trusted execution environments of the Obscuro network. It can not be seen by any node computer or by any node operator. Each guess costs 1 token. If you guess incorrectly, your fee is added to the prize pool. But if you guess correctly, you'll win all of the prize pool! Check the docs here for more info.
